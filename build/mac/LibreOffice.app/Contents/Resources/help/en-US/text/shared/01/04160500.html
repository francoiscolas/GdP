Insert Floating Frame

Inserts a floating frame into the current document. Floating frames are used in HTML documents to display the contents of another file.

To access this command...

Choose Insert - Floating Frame.

On the Insert bar, click

Floating Frame


If you want to create HTML pages that use floating frames, choose LibreOffice - PreferencesTools - Options - Load/Save - HTML compatibility, and then select the "MS Internet Explorer" option. The floating frame is bounded by <IFRAME> and </IFRAME> tags.
