Agenda Wizard - Agenda Items

Specifies the topics to be printed on the agenda template.

To access this command...

Choose File - Wizards - Agenda - Agenda items.


Topics

Enter the agenda topics. Use the Move up and Move down buttons to sort the topics.

Insert

Inserts a new empty topic row above the current row.

Remove

Removes the current topic row.

Move up

Moves the current topic row up.

Move down

Moves the current topic row down.
