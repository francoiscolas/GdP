YEAR

Returns the year as a number according to the internal calculation rules.

Syntax

YEAR(Number)

Number shows the internal date value for which the year is to be returned.

Example

=YEAR(1) returns 1899

=YEAR(2) returns 1900

=YEAR(33333.33) returns 1991
