Applying Text Formatting While You Type

To apply bold formatting

  1. Select the text that you want to format.

  2. Press Command Ctrl+B.

    You can also press Command Ctrl+B, type the text that you want to format in bold, and then press Command Ctrl+B when you are finished.

To apply italic formatting

  1. Select the text that you want to format.

  2. Press Command Ctrl+I.

    You can also press Command Ctrl+I, type the text that you want to format in italic, and then press Command Ctrl+I when you are finished.

To underline text

  1. Select the text that you want to underline.

  2. Press Command Ctrl+U.

    You can also press Command Ctrl+U, type the text that you want underlined, and then press Command Ctrl+U when you are finished.
