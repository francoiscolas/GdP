Bullets and Numbering Bar

The Bullets and Numbering bar contains functions to modify the structure of numbered paragraphs, including changing the order of paragraphs and defining different paragraph levels.

Numbering Off

Turns off numbering or bullets in the current paragraph or selected paragraphs.

Numbering Off

Promote One Level Promote

Moves the selected paragraph up one level in the numbering or bullets hierarchy.

Promote One LevelPromote

Demote One Level Demote

Moves the selected paragraph down one level in a numbering or bullets hierarchy.

Demote One LevelDemote

Promote One Level With Subpoints

Shifts paragraphs with subpoints up one numbering level. This is only visible when the cursor is positioned within numbered or bulleted text.

Promote One Level With Subpoints

Demote One Level With Subpoints

Shifts paragraphs with subpoints down one level. This is only visible when the cursor is positioned within numbered or bulleted text.

Demote One Level With Subpoints

Insert Unnumbered Entry

Inserts a paragraph without numbering. The existing numbering will not be affected.

Insert Unnumbered Entry

Move Up

Positions the selected paragraph before the one above it.

Move Up

Move Down

Positions the selected paragraph after the one below it.

Move Down

Move Up with Subpoints

Moves a paragraph with subpoints to above the previous paragraph. This is only visible when the cursor is positioned within numbered or bulleted text.

Move Up with Subpoints

Move Down with Subpoints

Moves a paragraph with all its subpoints below the following paragraph. This is only visible when the cursor is positioned within numbered or bulleted text.

Move Down with Subpoints

Restart Numbering

Restarts the text numbering. This is only visible when the cursor is positioned within numbered or bulleted text.

Restart Numbering

Bullets and Numbering

Adds numbering or bullets to the current paragraph, and lets you edit format of the numbering or bullets.

Bullets On/Off
