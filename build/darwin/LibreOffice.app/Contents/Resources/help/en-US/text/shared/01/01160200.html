E-mail Document

Opens a new window in your default e-mail program with the current document as an attachment. The current file format is used. If the document is new and unsaved, the format specified in LibreOffice - PreferencesTools - Options - Load/Save - General is used.

To access this command...

Choose File - Send - E-mail Document.

E-mail Document


If the document is in HTML format, any embedded or linked images will not be sent with the e-mail.
